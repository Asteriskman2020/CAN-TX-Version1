================================================================================
  Notes
================================================================================
- Source: Benz S320 (W220) — last 20% of d2.csv (4603 frames, 33 unique IDs)
- Bus speed: 500 kbps
- All IDs are standard 11-bit (not extended)
- 0xFF bytes in SAM modules (0x3E3–0x3EB) = no faults / idle state
- OBD-II: 0x7DF=broadcast request, 0x7E8=ECM reply, 0x7E9=TCM reply
- PID 0x05 (coolant temp): value 0x86=134→94°C, value 0x82=130→90°C (val-40)
- PID 0x10 (MAF sensor): requested via 0x7DF
